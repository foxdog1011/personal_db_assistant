# 背景 (Background)

## 問題
現代人每天接觸大量資訊，但常見痛點包括：
- 資訊分散在不同平台（記事本、瀏覽器、截圖、APP）。
- 資料存了卻難以快速找到。
- 太複雜的工具學習成本高，一般人懶得用。

## 目標
開發一款「個人資料庫助手 (Personal DB Assistant)」，具備：
1. **快速收集** → 能立即丟入資訊（文字、連結、圖片）。
2. **快速搜尋** → 能毫秒級找到需要的內容。
3. **低門檻** → 不需學習標籤/分類，即開即用。
4. **離線可用** → 本地優先，避免雲端依賴。

## 使用者
- 一般大眾：學生、上班族、研究人員。
- 對資訊管理有需求，但不想學太複雜系統。
